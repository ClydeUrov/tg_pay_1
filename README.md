# tg_pay_1

Телеграмм бот, принимающий ответы с помощью Flask и ngrok. Через данный телеграмм бот произведена возможность оплаты товаров с помощью платежной системы LiqPay.

#### Необходимые переменные окружения:
- TELEGRAM_BOT_TOKEN
- PUBLIC_KEY
- PRIVATE_KEY
